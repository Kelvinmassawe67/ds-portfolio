# Project 3: Retail Sales Performance Dashboard

## Business Problem
Sales managers need quick, intuitive insights into performance trends, regional breakdowns, and profitability to make data-driven decisions.

## Dashboard Panels
1. **KPI Cards** — Revenue, Profit, Margin, Orders (with YoY change)
2. **Monthly Revenue Trend** — 3-year comparison with seasonal patterns
3. **Regional Revenue** — Horizontal bar chart by region
4. **Category Revenue & Margin** — Dual-axis category analysis
5. **Quarterly Comparison** — YoY quarterly breakdown
6. **Profit Margin Distribution** — Regional margin spread

## Key Insights
- Electronics is the top revenue category
- November–December shows 30–50% seasonal lift
- North region leads in revenue
- Average profit margin: ~50%
- 2024 YoY revenue growth: +0.7%

## Run
```bash
python src/dashboard.py
# Output: visualizations/sales_dashboard.png
```
